安裝模組ang:cli (-g:全域環境)
安裝後會放到node_modules資料夾內
	npm install -g @angular/cli


根據package.json安裝套件
	npm install


⭐npm update
更新可更新的 node_modules (從package.json & package-lock.json引用)
從git clone下來後需要使用


⭐npm audit fix --force
自動修復 (執行npm update後可能會提示使用)


查看已安裝的包
	npm list --depth=0
